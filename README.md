-- Pasos utilizados para levantar un API con EXPRESS

1.- Requerimos contar con Node.js

2.- Iniciar proyecto
npm init -y

3.- Instalación de Express 
sudo npm install express

4.- Istalar nodemon  (Opciona)
 sudo npm install -g nodemon

5.- Correr servidor 
npm run dev
